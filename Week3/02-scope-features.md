# [ชื่อเกม] — Scope & Feature List

## Unique Selling Point (USP)

1. บริหารทรัพยากรของร้าน
2. event เหตุการณ์ที่เกิดขึ้นเเบบสุ่ม
3. การพูดคุยกับ NPC ในเเต่ละสถานะการที่ต่างกัน
4. ความเป็น fantasy ที่มีความหลากหลาย

## In Scope — สิ่งที่ "จะทำ" ในภาคการศึกษานี้

| # | Feature   | Priority | หมายเหตุ                                                                  |
| - | --------- | -------- | --------------------------------------------------------------------------------- |
| 1 | [Feature] | Must     | การพูดคุยกับ NPC ในเเต่ละสถานะการที่ต่างกัน |
| 2 | [Feature] | Should   | บริหารทรัพยากรของร้าน                                        |
| 3 | [Feature] | Could    | event เหตุการณ์ที่เกิดขึ้นเเบบสุ่ม                    |

## Out of Scope — สิ่งที่ "จะไม่ทำ"

- ระบบภายในร้านเเบบ top down - ไม่จำเป็น เนียงจากจะใช้เป็นรูปเเบบ point&click
- ระบบตบเเต่งภายในร้าน - เป็น feature ที่ไม่จำเป็นขนาดนั้น เเละยังต้องใช้เวลาในการทำเพิ่มจึงตดออก

## Risks & Assumptions

| ความเสี่ยง                 | ผลกระทบ           | แผนรับมือ                   |
| ------------------------------------ | ------------------------ | ------------------------------------ |
| ระบบเกมหลากหลาย       | อาจเยอะไป       | เฉลียงาน                     |
| ไม่เคยทำเเนวเกมนี้ | อาจทำไม่เป็น | ศึกษาหาข้อมูลเพิ่ม |
